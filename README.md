# JOJOUP 初中达人筛选工具

这是一个面向 JOJOUP 初中全科学习APP的达人筛选工作台，用于把抖音/巨量星图候选达人自动整理为可执行的投放建议。

## 当前能力

- 五维评分：近期传播、人群精准、商业能力、竞品空间、内容切角。
- 对抗性审查：明确哪些数据可直接获取、哪些只能推断、哪些不能假设。
- 数据置信度：缺星图ID、缺A3/投后、竞品高频、学生向内容等情况会自动降置信。
- CSV导入：支持按 `数据字段模板.csv` 导入星图/人工整理数据并自动推导评分。
- CSV导出：导出当前筛选结果、推荐等级、数据缺口和Brief建议。

## 使用方式

直接打开 `index.html`，或访问 GitHub Pages 发布地址。

第一版建议工作流：

1. 在巨量星图达人广场筛选教育/亲子/初中/学科相关达人。
2. 导出达人名单，按 `数据字段模板.csv` 补齐关键字段。
3. 在页面点击「导入名单」，生成候选池评分。
4. 根据A档、B档、C档和数据缺口安排邀约、测试或补数。

## 数据原则

这个工具不把粉丝量、单条爆款或竞品投放当成充分依据。首要目标是最大化预算内的有效学习APP转化概率：

`有效触达 × 人群匹配 × 信任强度 × 行动意愿 ÷ 成本 - 风险 - 数据不确定性`

因此，未授权或无法验证的数据只参与低置信度判断，不能直接作为放量依据。
