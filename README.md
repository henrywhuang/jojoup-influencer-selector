# JOJOUP 初中达人筛选工具

这是一个面向 JOJOUP 初中全科学习APP的达人筛选工作台，用于把抖音/巨量星图候选达人自动整理为可执行的投放建议。

## 当前能力

- 五维评分：近期传播、人群精准、商业能力、竞品空间、内容切角。
- 对抗性审查：明确哪些数据可直接获取、哪些只能推断、哪些不能假设。
- 数据置信度：缺星图ID、缺A3/投后、竞品高频、学生向内容等情况会自动降置信。
- 星图页面采集：你登录星图后，由 agent 读取当前页面可见的达人列表、卡片和详情信息。
- 清单导出：导出当前推荐结果、推荐等级、数据缺口和Brief建议。

## 使用方式

直接打开 `index.html`，或访问 GitHub Pages 发布地址。

第一版建议工作流：

1. 在巨量星图达人广场筛选教育/亲子/初中/学科相关达人。
2. 登录星图并打开达人广场、搜索结果、人群包结果或达人详情页。
3. 告诉 agent “开始抓取”，由 `tools/xingtu-visible-capture.js` 读取当前页面可见数据。
4. agent 用 `tools/score-xingtu-capture.mjs` 把采集结果转成候选排名。
5. 根据A档、B档、C档和数据缺口安排邀约、测试或补数。

## 采集边界

- 只读取当前登录账号在页面上已经可见的文本和链接。
- 不读取账号密码、Cookie、localStorage 或隐藏接口。
- 不绕过星图权限，不假设页面未展示的数据。
- 投后数据、A3、组件点击、应用下载等指标，只在账号有权限且页面展示时采集。

## 数据原则

这个工具不把粉丝量、单条爆款或竞品投放当成充分依据。首要目标是最大化预算内的有效学习APP转化概率：

`有效触达 × 人群匹配 × 信任强度 × 行动意愿 ÷ 成本 - 风险 - 数据不确定性`

因此，未授权或无法验证的数据只参与低置信度判断，不能直接作为放量依据。
